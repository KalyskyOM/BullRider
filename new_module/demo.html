The dashboard now includes an interactive performance chart modal. Here's how to use it:

  1. Navigate to the main dashboard
  2. Find the "Performance Trend" card (third achievement card with chart icon)
  3. Click on the Performance Trend card to open the chart modal
  4. View Mael's performance data with two data series:
    • Green line: % from category winner
    • Yellow/Brown line: % from fastest male
  5. Click legend items to toggle data series on/off
  6. Hover over data points for detailed information
  7. Close modal by clicking the X button, outside the modal, or pressing Escape
